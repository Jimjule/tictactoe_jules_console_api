# TicTacToe-Ruby

A command line API for my Tictactoe Gem.

# Installation

- Clone this repo to your directory
- Run `bundle install` in the project root

# Run

- Navigate to the project root
- Run `ruby app.rb`

# Testing

- Navigate to the project root
- Run `rspec`
- Tests are located in the 'spec' folder, and end in 'spec.rb'
